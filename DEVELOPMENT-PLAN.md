# Development Plan

## Goal
Create a mobile-first wrestling event discovery app using the current JSON schedule data.

## Phase 1: Foundation
- Set up React + Vite project
- Create the initial folder structure
- Add the architecture brief and development plan
- Configure testing tools

## Phase 2: Data and Utilities
- Load the schedule JSON data
- Create utility functions for filtering and formatting
- Add tests for the utility logic

## Phase 3: Core UI
- Build the weekly event list
- Create reusable event cards and search/filter components
- Add mobile-first styling

## Phase 4: Navigation and Detail Views
- Add week navigation
- Add event detail page
- Ensure the app remains small and component-driven

## Phase 5: Testing and Quality
- Add tests for user-facing behavior
- Improve coverage for search, filters, and rendering
- Refactor any large or mixed-responsibility files

## Definition of Done for Each Feature
- Code is split into small components or utilities
- Related tests exist and pass
- The functionality works on mobile layouts
- The implementation follows the architecture brief
